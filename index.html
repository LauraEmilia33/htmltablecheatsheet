The < table > box establishes a table formatting context. Elements inside the < table > generate rectangular boxes. Each box occupies a number of table cells according to the following rules: The row boxes fill the table in the source code order from top to bottom. Each row box occupies one row of cells. A row group box occupies one or more row boxes. Column boxes are placed next to each other in source code order. Depending on the value of the dir attribute, the columns are laid in left-to-right or right-to-left direction. A column box occupies one or more columns of table cells. A column group box occupies one or more column boxes. A cell box may span over multiple rows and columns. User agents trim cells to fit in the available number of rows and columns.
